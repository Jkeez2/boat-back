# Boat App

This project represents the Boat App's API.
It is used by the front-end project to make CRUD operations.

## Setup

This project uses an embedded H2 database, you can manually create
a datasource, depending on your IDE. Datasource information here : src/main/resources/application.properties

Then, to initialize it you can run this script : src/main/resources/data.sql
